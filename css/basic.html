01. 내부 스타일 시트

CSS를 <style> 태그로 기재하여 HTML 문서 안에 저장합니다.
스타일이 기술된 페이지에만 적용됩니다.

<style type = "text/css">
p { color: red ; } </style>

02. 내부 스타일 시트

스타일 속성들을 .css 확장자를 가진 별도의 파일로 저장하고, 저장된 속성을 HTML 문서에 파일명으로 연결합니다.
<link> 태크를 이용하여 선언합니다.
스타일을 선언한 모든 HTML 페이지에 적용됩니다.

<link rel="stylesheet" type="text/css" href="style.css">

css 파일의 상단에는 html 문서에 지정한 동일한 문자셋을 지정합니다.

@charset "utf-8";

03. @import

공통으로 들어가야 할 스타일을 따로 저장하여 다른 CSS에 삽입시켜 사용할 수 있도록 돕습니다.
CSS 파일을 너무 여러 개로 분리하면 복잡해져 유지보수하기 힘들기에, 하나의 파일로 저장하는 것이 좋습니다.

CSS의 맨 윗줄에 기술하여 CSS 안으로 다른 CSS 파일을 불러 옵니다.

<style type="text/css">
@import url("style.css");
</style>

어떤 CSS 파일 안에 공통 CSS를 불러옵니다.

@charset "utf-8";
@important url("common.css");

04. 인라인 스타일(inline style)

태그에 CSS를 직접 정의해주는 기법.
내부 외부 스타일에서 기술된 속성보다 우선으로 적용되므로 우선순위가 절대적으로 높아야 할 경우에 사용할 수 있습니다. 스타일을 공통 CSS로 수정할 수 없고 일일이 html을 열어서 편집해야 합니다.

<body>
<p style="color: red ;"> 우리를 기쁘게 하는 것들. </p>
</body>
