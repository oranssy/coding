01. 블록 레벨 요소 (Block-level Element)

블록 레벨 요소는 출력 화면상에서 줄이 바뀌는 특성을 가지고 있습니다.

<h2> 블록 레벨 요소 </h2>
<h2> 블록 레벨 요소 성질 </h2>
<p> 줄 바꿈이 생깁니다 </p>

블록 레벨 요소
블록 레벨 요소 성질
줄 바꿈이 생깁니다


02. 인라인 요소 (Inline Element)

인라인 요소는 줄 바꿈 특성이 없기에, 화면상 행이 바뀌지 않고 한 줄로 출력됩니다.

<strong> 인라인 요소 </strong>
<span> 인라인 요소는 한 줄로 출력됩니다. </span>

인라인 요소 인라인 요소는 한 줄로 출력됩니다.


03. 블록 레벨 요소와 인라인 요소의 특성

특성 정리

요소 유형 특성
블록 레벨 요소 줄 바꿈이 일어납니다.
문자(Text)와 인라인 요소를 자식 요소로 포함할 수 있습니다.
블록 레벨 요소 중에는 블록 레벨 요소를 자식 요소로 포함할 수 있는 요소와 포함할 수 없는 요소가 있습니다.
인라인 요소 줄 바꿈이 일어나지 않습니다.
문자(Text)와 인라인 요소를 자식 요소로 포함할 수 있습니다.
블록 레벨 요소를 자식 요소로 포함할 수 없습니다.

콘텐츠 카테고리 (콘텐츠 모델)

콘텐츠 종류 내용 해당 요소
메타데이터 콘텐츠
(Metadata Content)		 <head> 태그 안에 포함되는 요소 : 콘텐츠의 표현, 동작 설명, 문서간 관계를 설정, 정보전달을 포함하는 요소 <base>, <link>, <meta>, <noscript>, <script>, <style>, <title>, ...
플로우 콘텐츠
(Flow Content)		 <body> 안에 포함되는 대부분의 요소 : 내용 흐름에 관한 요소 <h1>, <h2>, <h3>, <h4>, <h5>, <h6>, <address>, <p>, <a>, <img>, <ul>, <ol>, <dl>, <table>, <form>, ...
섹션 콘텐츠
(Sectioning Content)		 heading과 footer의 범위를 정의하는 요소 <article>, <aside>, <nav>, <section>
헤딩 콘텐츠
(Heading Content)		 섹션의 헤더를 정의하는 요소 <h1>, <h2>, <h3>, <h4>, <h5>, <h6>
프레이징 콘텐츠
(Phrasing Content)		 문자(Text)를 마크업하는 요소 <a>, <em>, <strong>, <sub>, <sup>, <label>, ...
임베디드 콘텐츠
(Embedded Content)		 HTML 문서에 다른 리소스를 삽입하는 요소 <img>, <audio>, <video>, <iframe>, ...
인터랙티브 콘텐츠
(Interactive Content)		 사용자의 상호작용을 위한 요소 <a>, <button>, <audio>, <video>, <select>, ...
