레이아웃이란?

레이아웃(Layout, 배치)은 디자인에서 각 구성요소를 제한된 공간 안에 효과적으로 배열하는 일을 의미합니다. 레이아웃은 활동 또는 앱 위젯의 UI와 같은 사용자 인터페이스의 시각적 구조를 정의합니다.

레이아웃의 기본

<div> 박스 : block 구조

01. float 활용

float는 '둥둥 띄우다'라는 뜻이며, 글씨와 이미지를 정렬하는 기능을 합니다. float의 기능을 레이아웃에 활용할 수 있으나, 약간의 버그가 발생합니다.

02. flex 활용

flex는 레이아웃을 배치하는 기능이며, 가로 방향으로 배치됩니다.

03. 들여쓰기

자바스크립트는 가독성을 위하여 들여쓰기를 사용합니다.

04. 주석

자바스크립트는 주석은 일반적으로 특정 코드에 대한 설명을 입력하거나 특정한 부분의 코드가 실행되지 않도록 사용합니다. 주석 표시는 프로그램에 영향을 미치지 않습니다.

출력하는 방법

자바스크립트에서는 출력할 수 있는 방법이 여러가지가 있습니다.

분류 메서드
입력 prompt( )
출력 document.write( )
alert( )
console.log( )
확인 confirm( )

자바스크립트 사용 방법

자바스크립트의 사용 방법은 1. 외부파일 연결하는 방법 2. 스크립트 태그에 사용하는방법 3. 태그에 직접 기술하는 방법 등이 있습니다.
