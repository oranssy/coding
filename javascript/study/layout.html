레이아웃이란?

레이아웃(Layout, 배치)은 디자인에서 각 구성요소를 제한된 공간 안에 효과적으로 배열하는 일을 의미합니다.
레이아웃은 활동 또는 앱 위젯의 UI와 같은 사용자 인터페이스의 시각적 구조를 정의합니다.

레이아웃의 기본

<div> 박스 : block 구조

01. float 활용

float는 '둥둥 띄우다'라는 뜻이며, 글씨와 이미지를 정렬하는 기능을 합니다.
float의 기능을 레이아웃에 활용할 수 있으나, 약간의 버그가 발생합니다.
공간이 채워져 있다는 것을 설정하기 위해 clear : both; 를 사용합니다.

See the Pen Untitled by oranssy (@oranssy) on CodePen.

02. flex 활용

flex는 레이아웃을 배치하는 기능이며, 가로 방향으로 배치됩니다.

부모-자식 관계에서 유지가 됩니다.
기본 태그를 시멘틱 태그로 바꿔줍니다.
div 태그에 이름을 지어 의미를 부여합니다.
display:flex; // block 성질 변화시키기 :가로로 정렬하도록
flex-wrap:wrap; // 자식요소의 치수에 맞춰 작업해주기 위함
IE 버전에서는 적용이 되지 않음

03. grid 활용

flex에서 더 발전한 단계.
table을 나눠서 생각해봅니다.
display:grid; // 부모가 알아서 조절해주도록
grid-template-areas: // 나눠서 이름 지정함
grid-template-columns; // 가로 '열'의 비율 지정
grid-template-rows; // 세로 '행'의 비율 지정
