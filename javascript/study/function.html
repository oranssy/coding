함수

함수(Function)는 프로그램에서 반복적으로 사용되는 기능을 만들어 내기 위한 코드들의 집합을 나타내는 자료형 입니다.
함수를 사용하는 것을 '함수를 호출한다'라고 표현하며, 함수를 호출할 때 전달하는 자료를 '매개변수', 함수를 호출해서 최종적으로 나오는 결과를 '리턴값'이라고 합니다.

함수를 사용하면, 반복되는 코드를 함 번 정의해놓고 필요할 때마 호출하하여 반복 작업을 최소화 할 수 있습니다.
또한 기능에 따라 수정이 가능하므로 유지 보수가 용이하며, 긴 프로그램을 여러 함수의 작성을 통해 모듈화하여 가독성이 좋습니다.

함수의 종류

01. 선언적 함수

이름을 선언한 함수를 말합니다.

선언적 함수식


function func(){
    document.write("함수 실행 01 
");       /* 함수 선언 */
}
func();         /* 함수 호출 */

02. 익명 함수

이름이 붙어 있지 않은 함수를 말합니다.

익명 함수식


const func = function(){
    document.write("함수 실행 02 
");
}
func();

03. 매개변수 함수

매개변수 = 파라미터 = 인자
모든 함수에 매개변수와 리턴값을 사용하지 않고 필요한 경우에만 사용합니다.
매개변수를 사용하여 임의의 데이터를 함수 안에 전달합니다.
매개변수에 값을 전달하지 않으면 그 값은 'undefined'가 됩니다.

매개변수 함수식


function func(str){
    document.write(str);
    /*  return str; 이 생략되어 있음. */
}
func("함수 실행 03 
");

04. 리턴값 함수

리턴값이란 함수가 어떤 기능을 수행하고 그 결과를 호출한 곳으로 돌려주는 값을 말합니다.

리턴값 함수식


function func(){
    const str = "함수가 실행되었습니다.4 
"
    return str;
    /* document.write(str); 원래는 써야 하는데 쓰기 귀찮아서 안 쓰고 */
    /* 리턴값 함수는 return 값은 생략하지 않고 써줘야 함 */
    /* 결국 밑의 결과 값에 사용. */
}
document.write(func());

05. 화살표 함수 (선언적 함수)

화살표 함수는 ECMAscript6(es6) 문법으로, 화살표(=>)를 사용하여 함수의 기본 형식을 간결하게 표현하는 방법을 말합니다.
선언적 함수를 화살표 함수로 바꿉니다.

화살표 함수식 (선언적)


func = () => {
    document.write("함수가 실행되었습니다.5 
");
}
func();
    }      /* 이를 한 줄로 쓰면 아래와 같다. */

func = () => document.write("함수 실행 05 
");

func();

07. 화살표 함수 (매개변수 함수)

매개변수 함수를 화살표 함수로 바꿉니다.

화살표 함수식 (매개변수)


func = (str) => {         /* 매개변수 함수에서는 변수 선언이 생략 가능 */
    document.write(str);
}

func("함수 실행 07 
");

func = (str) => document.write(str);


func = str => document.write(str);      /* 값이 하나면 (str) 괄호 생략 가능*/
    
func("함수 실행 07 
");
