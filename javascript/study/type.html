데이터 타입 (자료형)

데이터 타입(Data Type)은 변수에 저장되는 데이터의 유형을 의미합니다.

01. 숫자 데이터

숫자(number) 데이터는 정수, 소수점, 지수를 표현할 수 있습니다.

var num1 = 10;
var num2 = 10,5;
var num3 = 1e+2;

02. 문자 데이터

문자(string) 데이터는 '' 또는 ""으로 표현할 수 있습니다. 보통 문자 데이터는 작은 따옴표로 표현합니다. 문자 안에 문자가 들어갈 경우, 바깥쪽을 작은 따옴표로 표현하였다면 안쪽을 큰 따옴표로 표현합니다.

var str1 = '문자';
var str2 = "문자";
var str3 = ' 문자는 "문자" ';
// 문자 안에 문자가 들어갈 경우 var str4 = " 문자는 '문자' ";
var str5 = "1";

특수한 목적을 위해 Escape Sequence '\(역슬래시)' 문자를 사용하는 경우도 있습니다. document.write()에서는 <br> 태그로 행을 바꿀 수 있지만 alert()에서는 바꿀 수 없는데, 이와 같은 경우에 '\n'을 사용하면 행을 바꿀 수 있습니다.

alert('자바스크립트\n제이쿼리');
자바스크립트
제이쿼리
이스케이프 문자 설명
\n 행 바꿈
\t 탭 문자
\\ 역슬래시
\' 작은 따옴표
\'' 큰 따옴표

03. 논리 데이터

논리(boolean) 데이터는 true(참)와 false(거짓)의 값을 표현합니다.

var temp1 = (5 > 4);
var temp2 = (5 < 4);
console.log(temp1); // true
console.log(temp2); // false

논리(boolean) 데이터에서 0값은 false를 의미합니다. 0값 이외의 숫자나 문자 값은 true를 의미합니다.

var temp1 = 0;
// false var temp2 = 1;
// true

04. undefined 데이터

undefined 데이터는 변수는 선언하였지만 데이터 값을 지정하지 않았을 경우, 객체의 속성 값ㅇ르 지정하지 않았을 경우에 자동으로 저장되는 값입니다.

var obj = {};
obj.name = '홍길동';
obj.age;
console.log(obj.age); undefined

05. null 데이터

null 데이터는 변수를 빈(empty) 상태로 초기화 하거나, 데이터를 저장하였으나 값이 존재하지 않을 때 null을 저장합니다.

var obj = 10;
obj = null;
console.log(obj); null

06. typeof 명령

typeof 명령을 사용하여 변수에 저장되어 있는 데이터의 타입을 알아볼 수 있습니다.

var num = 10;
var str = '문자';
console.log(typeof num); number
console.log(typeof str); string
