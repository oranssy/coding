지역 변수와 전역 변수

변수는 '함수 블록 { }'을 기준으로, 변수의 선언 위치에 따라 '지역 변수'와 '전역 변수'로 나뉩니다.

01. 지역 변수

지역 변수는 하나의 지역 내에서만 사용할 수 있는 변수를 뜻하며, 여기서 지역이란 '함수 블록 { }'에 의해 만들어지는 영역을 말합니다.

function 함수 ( )  { 
     var 변수 ;          // { } 안에서 var로 지역함수 선언
 } 

지역 변수의 변수 선언은 함수 블록 { } 안에서 이루어집니다.
함수의 지역 변수는 함수 블록 { } 안에서만 사용할 수 있습니다.
지역 변수는 함수가 실행되면 만들어지고 함수가 종료되면 소멸합니다.
함수의 외부에서는 지역 변수에 접근할 수 없습니다.
지역 변수는 함수 블록 { } 밖의 함수와는 다른, 별도의 변수가 됩니다. 

{
    var x = 100;             //변수 x에 숫자 100을 저장함
    var y = 200;             //변수 y에 숫자 200을 저장함
    var z = "javascript"    //변수 z에 문자열 "javascript"를 저장함

    document.write(x);
    document.write(y);
    document.write(z);
}

02. 전역 변수

전역 변수는 모든 영역(지역)에서 사용할 수 있는 변수를 뜻하며, '함수 블록 { }' 내외에서 자유롭게 사용 가능합니다.

var 변수;                  // { } 밖에서 전역 변수 선언
function 함수 ( )  { 
 } 

전역 변수의 변수 선언은 함수 블록 { } 밖에서 이루어집니다.
전역 변수는 함수 블록 { } 내외에서 모두 사용할 수 있습니다.
전역 변수는 웹 페이지가 닫혀야 사라집니다.
전역 변수는 프로그램의 어느 영역에서든 접근할 수 있습니다.

변수의 scope

블록 { }에 의해 변수의 범위가 달라지는 것을 말합니다.

전역 변수는 블록 { } 내외에서 모두 사용 가능하기에 자칫 동일한 이름으로 덮어 써질 수 있는 문제점이 있습니다.
따라서 전역 변수 보다 지역 변수가 더 안전합니다.

수

01. 세미콜론(;)

하나의 실행문이 끝나면 마지막에 세미콜론(;)을 붙여 줍니다. 세미콜론(;)을 붙여 주지 않아도 상관은 없습니다.

02. 대소문자

자바스크립트는 명령어는 대소문자를 구별합니다. console과 Console은 각각 다른 문자열로 처리됩니다.

03. 들여쓰기

자바스크립트는 가독성을 위하여 들여쓰기를 사용합니다.

04. 주석

자바스크립트는 주석은 일반적으로 특정 코드에 대한 설명을 입력하거나 특정한 부분의 코드가 실행되지 않도록 사용합니다. 주석 표시는 프로그램에 영향을 미치지 않습니다.

출력하는 방법

자바스크립트에서 출력하는 방법은 여러가지 입니다.

분류 메서드 (method)
입력 prompt( )
출력 document.write( )
alert( )
console.log( )
확인 confirm( )

자바스크립트 사용 방법

자바스크립트의 사용 방법은 1. 외부파일 연결하는 방법
2. 스크립트 태그에 사용하는방법
3. 태그에 직접 기술하는 방법 등이 있습니다. 

<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <title>Javasript</title>

    <!-- 1. 외부 파일로 로드하는 방법 -->
    <script type="text/Javasript" src="js/script.js"></script>

    <!-- 2. script 태그 사이에 기술하는 방법 -->
    <script>
        document.write("Javasript");
    </script>
</head>
<body>

    <!-- 3. 태그에 직접 기술하는 방법 -->
    <a href="#" onclick="alert('Javasript')"></a>
</body>
</html>
